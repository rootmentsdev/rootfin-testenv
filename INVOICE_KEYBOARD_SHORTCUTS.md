# Invoice Creation - Keyboard Shortcuts

## Available Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+O** | Open New Invoice | Creates a new invoice (navigates to create page) |
| **Ctrl+N** | New Invoice | Same as Ctrl+O - creates a new invoice |
| **Ctrl+I** | Invoice List | Opens the invoices list page |

## How to Use

### Ctrl+O - Open New Invoice
**When to use:** When you want to quickly create a new invoice
**How to use:** Press and hold `Ctrl` key, then press `O`
**Result:** 
- If on create page: No action (already creating)
- If on edit page: Navigates to new invoice creation page

### Ctrl+N - New Invoice
**When to use:** Quick shortcut to create a new invoice
**How to use:** Press and hold `Ctrl` key, then press `N`
**Result:** Same as Ctrl+O - creates a new invoice

### Ctrl+I - Invoice List
**When to use:** When you want to view all invoices
**How to use:** Press and hold `Ctrl` key, then press `I`
**Result:** Navigates to the invoices list page

## Example Workflows

### Workflow 1: Create Multiple Invoices
```
1. Press Ctrl+O to open new invoice page
2. Fill in invoice details
3. Save invoice
4. Press Ctrl+O again to create another invoice
```

### Workflow 2: Quick Navigation
```
1. Press Ctrl+I to view all invoices
2. Click on an invoice to edit
3. Press Ctrl+O to create a new invoice
4. Press Ctrl+I to go back to list
```

### Workflow 3: Edit and Create
```
1. Press Ctrl+I to open invoice list
2. Click on invoice to edit
3. Make changes and save
4. Press Ctrl+N to create a new invoice
```

## Tips & Tricks

### For Power Users
- Use Ctrl+O or Ctrl+N to quickly switch between creating new invoices
- Use Ctrl+I to quickly navigate to the invoice list
- Combine with Tab key to navigate between form fields
- Use Enter to submit forms (when available)

### Accessibility
- All keyboard shortcuts are accessible
- No special keys required (standard Ctrl modifier)
- Works on Windows, Mac (Cmd instead of Ctrl), and Linux

### On Mac
- Use `Cmd+O` instead of `Ctrl+O`
- Use `Cmd+N` instead of `Ctrl+N`
- Use `Cmd+I` instead of `Ctrl+I`
- The system automatically handles this

## Troubleshooting

### Shortcut not working?
1. Make sure you're on the invoice page
2. Check that Ctrl key is pressed (not Cmd on Mac)
3. Ensure the page has focus (click on the page first)
4. Try refreshing the page
5. Check browser console for errors

### Conflict with browser shortcuts?
- Some browsers may have default shortcuts that conflict
- Try using the shortcut in a different context
- Check your browser's keyboard shortcut settings
